# Subagentes Paralelos

## Frentes usadas nesta entrega

- `subagente-dados`: mapeamento do esquema, escrita das consultas SQL e verificacao de disponibilidade no PostgreSQL
- `subagente-ui`: traducao das 9 imagens para 9 abas Streamlit
- `subagente-lacunas`: identificacao de dados faltantes, criacao de placeholders e documentacao dos downloads do TSE

## Como evoluir

Se o app crescer, mantenha essas tres frentes separadas:

- dados
- interface
- ingestao e lacunas
